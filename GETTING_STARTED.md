## 🚀 Quick Start

### Step 1: Install Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Step 2: Install Frontend
```bash
cd frontend
npm install
npm run dev
```

Open: `http://localhost:5173`
